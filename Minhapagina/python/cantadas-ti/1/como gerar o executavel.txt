primeiro instaler o cx_freeze

execute no CMD  ou Direto no terminal do VS pip install cx-freeze

depois crie um arquivo setup.py na mesma pasta que o arquivo que deseja gerar o executavel.

abra o setup.py e dentro de executable coloque o nome do arquivo que deseja gerar o excutável
caso queira incluir imagens ou outra coisa informe no buildoptions

nomeie os campos no setup (linha 23)

em seguida vá na pasta aonde esta o seu programa e o setup e abra o terminal

ou abra o terminal e digite: cd (endereço da pasta)

digite:  python setup.py build    aperte enter e aguarde criar o arquivo



caso tenha perdido o arquivo setup.py o codigo dele está aqui em baixo

=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
				CODIGO SETUP.PY
=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-=-
import sys
from cx_Freeze import setup, Executable


base = None
if sys.platform == "win32":
    base = "Win32GUI"

executables = [
        Executable("script.py", base=base)
]

buildOptions = dict(
        packages = [],
        includes = [],
        include_files = [],
        excludes = []
)




setup(
    name = "NomeDoPrograma",
    version = "1.0",
    description = "Descrição do programa",
    options = dict(build_exe = buildOptions),
    executables = executables
 )
